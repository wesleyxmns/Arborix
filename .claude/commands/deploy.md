# Deploy Arborix

Execute o fluxo completo de deploy do Arborix:

1. Criar nova branch de feature
2. Commitar todas as mudanças com mensagens detalhadas
3. Push para o GitHub
4. Criar Pull Request detalhado
5. Fazer merge na main
6. Atualizar versão no package.json (solicitar versão ao usuário)
7. Criar release no GitHub
8. Build do projeto
9. Publicar no npm

**IMPORTANTE:** Antes de executar, certifique-se de:
- Todas as mudanças estão prontas para commit
- Você está na branch correta
- Testes passaram (se houver)
- Build está funcionando

O assistente irá guiá-lo pelo processo, solicitando informações quando necessário (como versão e nome da feature).
